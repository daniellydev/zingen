# Zingen

Landing page desenvolvida com HTML5 e CSS3 para divulgar um aplicativo de karaokê. O projeto foi criado com foco na construção de interfaces modernas, responsividade e aplicação de boas práticas de desenvolvimento Front-end.

## Sobre o projeto

O **Zingen** é uma landing page desenvolvida para apresentar um aplicativo de karaokê, destacando seus principais recursos, benefícios e chamada para download.

Durante o desenvolvimento foram aplicados conceitos de organização de layout, hierarquia visual, responsividade e estilização utilizando HTML5 e CSS3.

## Tecnologias utilizadas

- HTML5
- CSS3
- Git
- GitHub
- GitHub Pages

## Funcionalidades

- Página inicial responsiva
- Seção de apresentação do aplicativo
- Destaque para funcionalidades
- Layout moderno e intuitivo
- Chamada para download do aplicativo
- Interface otimizada para diferentes tamanhos de tela

## Conceitos praticados

- Estrutura HTML semântica
- Flexbox
- CSS Grid
- Posicionamento de elementos
- Responsividade
- Variáveis CSS
- Tipografia
- Organização de componentes
- Efeitos visuais e transições

## Como executar o projeto

1. Clone este repositório:

```bash
git clone https://github.com/daniellydev/zingen.git
```

2. Acesse a pasta do projeto:

```bash
cd zingen
```

3. Abra o arquivo `index.html` em seu navegador.
